# برنامج الفواتير

نسخة جاهزة للرفع على GitHub Pages.

## التشغيل محليًا
افتح ملف `index.html` مباشرة من المتصفح.

## الرفع على GitHub
1. اعمل Repository جديد على GitHub.
2. ارفع الملفات الموجودة في هذا الفولدر.
3. من Settings > Pages اختار:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
4. افتح رابط GitHub Pages بعد النشر.

## ملاحظات مهمة
- البرنامج يعتمد على LocalStorage داخل المتصفح.
- البيانات محفوظة على نفس الجهاز والمتصفح.
- قبل استخدام زر THE END أو نقل البرنامج لجهاز آخر، اعمل نسخة احتياطية من داخل البرنامج.
- المكتبات الخارجية تعمل عن طريق CDN، لذلك الطباعة/PDF/Excel تحتاج إنترنت أول مرة أو أثناء الاستخدام حسب المتصفح.

## المميزات
- فواتير مبيعات.
- مرتجعات تزيد المخزون مرة أخرى.
- مشتريات ومخازن وجرد.
- موردين ومدفوعات.
- تقارير وتصدير Excel.
- طباعة فواتير المشتريات.
- تعديل فواتير المشتريات بباسورد.
